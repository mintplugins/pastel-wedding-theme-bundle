A bundle of Stack Templates, Plugins, a Theme, and everything needed for the Pastel Wedding Theme Experience.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the ‘pastel-wedding-theme-bundle’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
